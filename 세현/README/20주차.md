# 🧩 회고 문제

## [바이러스 ](https://www.acmicpc.net/problem/2606)

### 💡 아이디어 / 접근법

- 양방향 그래프를 인접 리스트로 저장
- bfs 돌면서 방문한 컴퓨터 감염(visited) 체크
- 체크된 수 합산(1번 컴퓨터 제외)

### 🤔 후기

- 그래프 너무 오랜만에 봐서 무방향 그래프 처리 까먹음
- 그래서 처음에 왼쪽 리스트에 오른쪽 노드 삽입만 하고 넘어감
- 무방향은 서로 인접리스트에 넣어 주는걸 잊지 말자

## [나이트의 이동](https://www.acmicpc.net/problem/7562)

### 💡 아이디어 / 접근법

- 델타 탐색 + bfs
- visited 배열 대신 dist 배열 사용(visited + 이동거리 저장)
- bfs 돌면서 dist 배열에 거리 표시
- 목적지 도달 시 반환

### 🤔 후기

- 처음에는 사이클 돌리면서 해당 사이클이 지나면 카운트 증가하는 식으로 품
- 더 단순하게 풀이: 아예 visited 배열에다 거리 정보까지 저장하는 방법이 정석임
- 너무 어렵게 꼬아서 푼 것 같음
- 기억해 놨다가 최단 경로 bfs 문제에 써야 겠다 생각함
