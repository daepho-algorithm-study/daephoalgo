## [나무 자르기](https://www.acmicpc.net/problem/2805)

### 💡 아이디어 / 접근법

- 이분탐색 + 매개변수 탐색
- mid 가 값에 해당되어도 계속해서 최댓값을 갱신해 나가야 함
- 탐색한 값이 작으면 left 를 mid + 1로, 크다면 right 를 mid - 1 로 (기본적인 이분탐색 풀이법)
- 정답 후보를 따로 저장해서 계속 갱신해 나가기

### 🤔 후기

- 처음엔 정답 값을 따로 저장하지 않고 mid 값을 갱신하면서 풀었더니 틀렸음
- 정답 값을 따로 저장해 줘야 정확함
