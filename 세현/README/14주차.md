## [영단어 암기는 괴로워](https://www.acmicpc.net/problem/20920)

### 💡 아이디어 / 접근법

- js의 sort를 적극 활용함
- js의 sort는 콜백함수를 인자로 넘겨줄 수 있음
- 콜백 함수는 파라미터로 두 인자를 받고 두 인자를 비교해서
- +가 나오면 순서를 바꾸지 않고 -가 나오면 순서를 바꿈
- 이 성질을 이용해서 조건문으로 세세하게 순서를 조정할 수 있음

### 🤔 후기

- js sort 쓰면서 python에서 sort쓸 때 key로 lambda 함수 지정하는 것과 비슷하다고 느낌
- js 쪽이 더 직관적으로 느껴졌음
