# 🧩 회고 문제

## [공원](https://school.programmers.co.kr/learn/courses/30/lessons/340198)

### 💡 아이디어 / 접근법

- DP 문제
- 깔 수 있는 돗자리 크기를 위치마다 계산
- 이전의 돗자리 크기 결과를 가져와서 더해주기
- 가장 큰 크기의 돗자리를 리턴하면 끝

### 🤔 후기

- 문제만 보고 dp인지 감을 잡지 못했음
- 1시간동안 삽질하다 결국 ai의 도움을 받음(dp라는 힌트를 받으니 문제가 쉬워짐)
- 알고리즘 유형을 안보고 풀어야 문제의 유형이 기억에 잘 남는 거 같음
