# 🧩 문제

## [풍선 터뜨리기](https://www.acmicpc.net/problem/2346)

### 💡 아이디어 / 접근법

- 덱을 이용해서 앞뒤로 움직이기
- 맨 앞을 떼서 맨 뒤에 붙이는 행위가 오른쪽으로 한 칸 가는 것 과 같음
- 맨 뒤를 떼서 맨 앞을 붙이는 행위가 왼쪽으로 한 칸 가는 것 과 같음


### 🤔 후기

- 다른 언어는 다 메모리를 적절히 쓸 수 있는데 node.js만 이상하게 가용 메모리가 적어서 시간이 매우 오래 걸렸던 문제
- 잘못 풀었다고 생각해 음수 나머지 개념(((m%k)+k)%k)까지 도입해 봤는데 결과적으로 헛수고가 되어버림
- 돌이켜 생각해보면 메모리 초과인데 시간을 줄이려고 했던 것 같음 (그냥 바보가 되어버림)
- 매우 허탈했던 문제 (백준 노드 차별 좀 그만)
