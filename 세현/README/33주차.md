## [가장 먼 노드](https://school.programmers.co.kr/learn/courses/30/lessons/49189)

### 💡 아이디어 / 접근법

- 양방향 그래프의 리프 노드 개수 찾기
- 루트는 1로 고정되어 있음
- 인접 노드 리스트를 만들어서 푸는게 가장 효율적

### 🤔 후기

- 처음에 인접노드 리스트 안만들고 간선정보 그대로 풀었더니 시간이 7000ms 인가 찍힘
- 백준에서 풀었으면 바로 틀렸겠다 싶어서 다른사람들 어떻게 풀었나 찾아봤더니 인접리스트 따로 만들어서 푼 것을 발견
- 그래프 문제 푼지 시간이 꽤 되어서 까먹었더니 모든 접근법이 브루트포스 + 플래그 제어 형식으로 회귀한 것 같음
- 자바로 푸니 시간이 배로 걸린거 같음
