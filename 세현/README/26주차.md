# 🧩 회고 문제

## [쿼드트리](https://www.acmicpc.net/problem/1992)

### 💡 아이디어 / 접근법

- 사분면으로 계속 나누면서 재귀 돌리기

### 🤔 후기

- 다 풀고 다른 풀이를 봤는데 N 하나 가지고 중간 부분을 연산으로 다 구함. 나는 구분하기 쉽게 연산 결과를 변수에다 담았음(x_start, x_end...)
- 카테고리가 분할 정복임을 모르는 상태에서 문제에 접근하면 이해하기 쉽지 않았을 것 같음(많이 풀어봐야 할듯)

---

## [곱셈](https://www.acmicpc.net/problem/1692)

### 💡 아이디어 / 접근법

- 거듭제곱 연산과 모듈러의 성질을 이용하는 문제
- a^n = (짝수) a^(n/2)*a^(n/2) | (홀수) a * a^((n-1)/2) * a^((n-1)/2)
- 모듈러 연산 결과는 중간에 계속 모듈러 해줘도 결과가 동일함

### 🤔 후기

- 수가 크면 BigInt 써야한다는 걸 자꾸 까먹음
- BigInt는 BigInt 끼리 나눠줘야 한다는걸 자꾸 까먹어서 많이 틀렸던 것 같음
