## [N번째 큰 수](https://www.acmicpc.net/problem/2075)

### 💡 아이디어 / 접근법

- N 크기의 minHeap 트리를 구현하기
- 힙의 루트 노드(가장 작은 수) 를 입력값과 계속 비교해 가면서 루트보다 크면 교체해주기
- 모든 처리가 끝나면 루트 노드가 N 번째 크기의 수가 됨

### 🤔 후기

- 처음에 브루트포스 방식으로 한번 시도해 봄(테케 통과) 근데 메모리 터짐
- 그래서 최소 힙 쓰는 방법으로 다시 해봄 -> 그래도 터짐.
- 그래서 파일 전체 다 읽은 후 입력 처리하는 방식에서 한줄 한줄 읽고 바로 처리하는 방식으로 변경 -> 그래도 터짐
- 구조분해 할당으로 스왑하던 거 임시 변수로 할당하는 방법으로 바꿨더니 그제서야 통과
- 한줄씩 읽고 바로 처리하도록 템플릿 바꿔야하나..
