# 🧩 후기 문제

## [체스판 다시 칠하기](https://www.acmicpc.net/problem/1018)

### 💡 아이디어 / 접근법
- 처음엔 8x8 크기라는걸 보지않고 예전 im 러시아국기? 문제처럼 체스판 칠하기만 하면 되는줄 알고 교차되게만 구현했음
- 수가 너무 많길래 문제를 다시보니 8x8 크기의 다시 색칠하면 되는 '최소값'을 구하는 문제
- 그래서 가로 세로로 8칸만 확인하고 다음칸으로 이동하며, 시작이 검정색일 경우와 백색인 경우 모두 탐색
- 다시 색칠해야하는 갯수를 체크해서 최소값을 구해서 찾아냈음

### 🤔 후기
- im 문제류(for와 if 만으로 푸는 문제)중 델타 제외하고 진짜 정석에 가까운 살짝 난이도 있는 문제
- 다들 한번 풀어보는것을 추천
