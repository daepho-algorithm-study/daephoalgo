# 🧩 문제

## [배달](https://school.programmers.co.kr/learn/courses/30/lessons/12978)

### 💡 아이디어 / 접근법

1. 양방향 그래프 만들기
2. 시작 마을 거리 0으로 초기화
3. 다익스트라 실행
4. 방문하지 않았고, 거리가 최소값이라면 최신화
5. 마을에서 다음 마을에 갈 때 필요한 최소값을 dist 배열에 저장
6. dist 배열 중 K 보다 작은 값들이 있으면 카운팅 후 반환

### 🤔 후기

문제를 보자마자 다익스트라로 풀어야겠다고 생각했다. 여러 노드들에서 최소비용을 체크할 필요 없고 하나의 노드(1번 마을)에서만 체크하면 되기 때문에 반복문 자체도 하나가 줄어들기 때문에 일단 기본 배열로 다익스트라를 짜봤는데 다행히 시간 초과 없이 잘 됐다
