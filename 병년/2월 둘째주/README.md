# 🧩 문제

## [택배상자](https://school.programmers.co.kr/learn/courses/30/lessons/131704)

### 💡 아이디어 / 접근법 
1. 일단 스택에 다 넣어
2. 스택 순회 하면서 스택의 마지막 값 확인해
3. 그게 지금 꺼내야 할 상자와 일치하면 상자 빼고 다음 꺼낼 상자 확인할 수 있도록 해
4. 아니라면 상자 빼지마

### 🤔 후기
처음에는 컨테이너 벨트에 넣고 넣은 순서대로 빼야할 줄 알고 queue 로 접근해서 1부터 order의 첫번째 값 까지 넣고 역순으로 만들었음 그리고 무조건 첫 번째 값은 뽑을 수 있도록 보장하도록 실행을 했는데 2가지 테케는 운 좋게 통과 했지만 히든 테케에서 계속 걸리더라.. 그래서 생각해보니까 queue의 역순 ? 그냥 stack으로 하면 되잖아? 하고 바꾸고 첫 번째 값을 무조건 뽑을 수 있도록 보장하는 것도 말이 안된다고 생각했음 또 이 상자를 뽑는 게 전체 다 쌓아놓고 뽑는 게 아니라 그때그때 상자에 넣고 뽑아서 확인을 해야 했어서 해당 코드로 수정했음.
