# 🧩 문제

## [아기상어2](https://www.acmicpc.net/problem/17086)

### 💡 아이디어 / 접근법

1. 반복문을 돌려서 상어의 좌표 다 넣어주기
2. 8방향 델타 탐색 이후 원본 배열 갱신 후 queue에 추가
3. 최댓값 출력

### 🤔 후기

다시 BFS 감을 잡고자 푼 문제인데 별로 어려운 것은 없었고 상어들의 좌표를 다 queue에 넣어놓고 시작해야 한다는 점이 좀 달랐음

**2.** [경쟁적 전염](https://www.acmicpc.net/problem/18405)

### 💡 아이디어 / 접근법

1. 0이 아닌 숫자 좌표 찾기
2. 정렬하여 우선순위 부여
3. 우선순위대로 BFS 돌리면서 새 배열에 기록하고 시간 + 1
4. 시간이 S 라면 반환

### 🤔 후기

해당 문제도 위의 아기상어 문제와 크게 다를 건 없었고 우선순위만 정해주면 되는 문제였음
