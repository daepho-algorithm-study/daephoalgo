# 🧩 문제

## [특정한 최단 경로](https://www.acmicpc.net/problem/1504)

### 💡 아이디어 / 접근법

### 플로이드 워셜 풀이

1. 대각선 값 0으로 초기화 (내 위치)
2. 입력 값에 따라 거리 배열에 dist[a - 1][b - 1] = c로 초기화
3. 플로이드 워셜을 돌려서 각 노드별로 최소 비용 저장
4. v1, v2 입력 받고
5. 맨 첫번째 위치에서 v1 가는 거리 + v1 에서 v2 가는 거리 + v2 에서 N 까지 가는 거리를 계산해서 특정 노드를 경유하며 끝 까지 가는 거리 구하기
6. 반대의 경우도 계산
7. 그 둘 중 최소 값이 정답

### 다익스트라 풀이

1. 간선, 비용 정보를 포함한 그래프 만들기
2. dist 배열믈 만들고 최소값 힙을 통해 우선순위 큐 다익스트라 실행
3. 처음 위치 0으로 초기화
4. 우선순위 큐에 비용과 처음 값 넣고 시작
5. 큐에서 뽑아와서 현재 뽑아낸 비용이 지금까지의 최소비용보다 더 크면 무시
6. graph[now] 에 대해 순회
7. 다음 비용을 현재 뽑은 비용에 합산
8. 다음 비용이 현재 저장되어 있는 비용보다 작으면 저장 큐에 삽입
9. v1, v2 입력 받고 시작점, v1, v2 각각 다익스트라 수행 후 최소 비용 배열 뽑아오기
10. 시작점부터 v1 까지의 비용, v1 에서 v2 까지의 비용, v2에서 N 까지의 비용의 합산과 반대의 합산을 비교해서 더 낮은 값이 정답

### 🤔 후기

익명의 제보로 문제를 한 번 풀어보라고 해서 풀어봤는데 보자마자 경유해야 하니까 플로이드 워셜을 생각했음 그래서 그렇게 문제를 풀었는데 근데 N이 800 이라서 5억이 넘는데 이게 시간초과가 안날까 싶었음 그렇게 제출했는데 ? 통과가 됨; 이유는 모르겠음 근데 이 문제는 이렇게 푸는 게 아닌 거 같아서 다익스트라로도 같이 풀어봄 ㅋ

## [순위](https://school.programmers.co.kr/learn/courses/30/lessons/49191?language=javascript)

### 💡 아이디어 / 접근법

1. dist 배열 정의 후 현재 값 0으로 초기화
2. results 순회하면서 dist 배열에 체크
3. 플로이드 와샬 돌리기
4. 2중 반복을 통해 dist[i][j], dist[j][i] 가 Infinity 가 아닐 경우 순서가 있는 것이기 때문에 count++
5. 해당 count가 n - 1 과 같다면 모든 순위를 알 수 있는 사람이기에 answer에 추가

### 🤔 후기

저번에 풀어 본 역사 문제와 동일한 문제여서 금방 풀었음. 얼핏 보면 위상정렬 같은데 이 문제는 정렬은 알 필요가 없고 순서를 정확히 알 수 있는 사람들만 필요하기에 프로이드 와샬로 풂 ㅋ

## [문자 해독](https://www.acmicpc.net/problem/1593)

### 💡 아이디어 / 접근법

1. 문자열 전체 길이와 부분 문자열의 길이 받기
2. 문자열 일치 여부를 판단하기 위해 알파벳 대소문자의 길이를 가지는 배열 2개 생성
3. 문자열 두 개 입력 받고 문자의 아스키 코드 변환 함수 수행
4. W(부분 문자열) 에 대해 순회, S(전체 문자열)에 대해 초기 부분 문자열의 길이만큼 알파벳 개수 초기 세팅
5. wCnt, gCnt를 탐색하면서 개수가 정확히 같으면 true 아니면 false 반환하는 isSame 함수 정의
6. 처음에 바로 isSame 함수를 통해 가능한 지 확인 그 후 4칸씩 이동하면서(슬라이싱 윈도우) isSame 함수를 통해 w,g 확인 후 같으면 answer++
7. answer 반환

### 🤔 후기

어제 새벽에 문제 풀다가 뭔가 되게 이해가 안되던 문제였음. 결과적으로는 전체 문자열에서 부분 문자열의 알파벳의 종류, 개수가 같은 지 계속 체크하면 되는 문제였는데...

## [카드 놓기](https://www.acmicpc.net/problem/5568)

### 💡 아이디어 / 접근법

1. 같은 숫자를 받지 않도록 set 정의
2. DFS 를통해 K 만큼 내려가면서 숫자 합치기
3. 해당 숫자 set에 넣고 return
4. set의 사이즈 출력해서 겹치지 않는 숫자의 개수 출력

### 🤔 후기

기본적인 백트래킹이 들어간 dfs 가 아닐까 생각하는데 순열을 뽑는
