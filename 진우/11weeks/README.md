# 🧩 문제

## [대칭 차집합 - 1269](https://www.acmicpc.net/problem/1269)

### 💡 아이디어 / 접근법
A와 B중 원소가 A 또는 B 원소에 포함되는 갯수만 확인하면 되기 때문에 하나만 집합으로 만들고 포함되는 원소의 갯수를 세어주면 된다.
대칭 차집합 개수 = |A| + |B| - 2 * |A ∩ B|이므로 겹치는 원소 갯수를 *2 해주면 된다.

### 🤔 후기
차집합이기 때문에 A에 포함되는건 B에도 포함되기 때문에 겹치는 원소를 2배 해주면 되는 문제

# 🧩 문제

## [나는야 포켓몬 마스터 이다솜 - 1620](https://www.acmicpc.net/problem/1620)

### 💡 아이디어 / 접근법
포켓몬 이름을 입력 받으면 도감의 번호를 번호를 입력 받으면 포켓몬 이름을 출력해야 한다. 
이름을 입력 받으면 번호를 출력하기 위해 map을 통해 key를 이름, value를 번호로 지정하고
vector를 통해 해당 인덱스 번호에 이름을 저장하여 번호로 접근 했을 때 이름을 출력할 수 있도록 한다.
만약 번호라면 문자열의 0번째 부분이 '0' ~ '9' 사이기 때문에 이 범위 안에 들어온다면 숫자로 판단하고 문자열을 정수로 변환한다. 

### 🤔 후기
문자열과 정수가 번갈아가며 입력되기 때문에 정수인 부분을 판별 해주어야 했다. 하지만 입력은 전부 string으로 받아오기에 이걸 구분할 로직을 생각하는 것이 어려웠다.
