# 🧩 문제

## [이분 그래프 - 1707](https://www.acmicpc.net/problem/1707)

### 💡 아이디어 / 접근법
양방향 그래프로 간선의 정보를 저장하고  BFS를 이용하여 이분 그래프를 판별하여 준다.
이분 그래프는 각 라인의 색이 같아야 하는것이므로 모든 노드를 순회하면서 
현재 노드와 연결된 모든 인접 노드 확인하고 방문하지 않은 노드라면, 현재 노드와 반대 색으로 칠해준다.
인접 노드가 현재 노드와 같은 색이면 이분 그래프 조건을 위배하므로 바로 함수를 탈출한다.

### 🤔 후기
이분 그래프에 대한 이해를 잘못했었다. 단순히 노드 하나에 간선이 3개이상 연결되면 쌍을 이루지 못하니까 이분 그래프가 안된다고 생각했는데 색으로 구분 했을 때 다른 색이어야 하는것이 이분그래프였다.

# 🧩 문제

## [벽 부수고 이동하기 - 2206](https://www.acmicpc.net/problem/2206)

### 💡 아이디어 / 접근법
일단 최단거리를 알아내야 하므로 bfs로 접근을 하였다.
방문 배열을 벽을 부쉈을 때와 부수지 않았을 때로 구분하기 위해 3차원 배열을 활용하여 구분하고,
탐색을 하면서 벽을 부수고 이동했을 때와 벽을 부수지 않고 이동했을 때 각각 다른 배열에 저장하고, 해당 벽의 값을 가져와 이전 값에서 +1을 더해준다.
결과적으로 벽을 부수지 않고 이동한 최단거리와 벽을 한번 부수고 이동한 최단거리 중 먼저 도착한 값을 출력하게 된다.

### 🤔 후기
정말 어려웠던 문제였다. 풀이를 보고도 이해가 잘 안됐던 문제

# 🧩 문제

## [뱀과 사다리 게임 - 16928](https://www.acmicpc.net/problem/16928)

### 💡 아이디어 / 접근법
어차피 뱀이나 사다리나 해당 칸에 도착하면 이동 하는것을 똑같기 때문에 board[x] = y로 간선 정보를 저장해준다.
100칸까지 가는 최소의 주사위 수를 구해야 하므로 bfs를 활용하였다.
각 칸까지 도달하는 최소 주사위 횟수 저장하면서 주사위 1~6까지 모든 경우를 탐색하여 준다. 
조건문을 통해 사다리 또는 뱀이 있으면 해당 위치로 이동하고, 아직 방문하지 않은 칸이면 현재까지 횟수 + 1를 해준다.
 
### 🤔 후기
어차피 사다리나 뱀이나 그 칸으로 이동을 하기 때문에 bfs를 활용하여 사다리나 뱀이 있는 경우와 없는 경우로 나누어 다음 칸을 큐에 넣고, 이전 배열에 값의 1을 더하면 되었다.
