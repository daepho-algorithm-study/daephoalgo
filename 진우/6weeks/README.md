# 🧩 문제

## [색종이 - 2563](https://www.acmicpc.net/problem/2563)

### 💡 아이디어 / 접근법 
- 첫 번째
-> 사각형 넓이에서 겹친 부분을 계산해서 뺄려고 했음. 하지만 로직 구현이 어려움

- 두 번째
-> 2차원 배열을 활용하여 색종이 범위를 전부 1로 채움

### 🤔 후기
도화지가 100 x 100이라 가능했던거 같다. 이차원 배열을 활용해서 전부 1로 채우면 쉽게 풀 수 있었다.
