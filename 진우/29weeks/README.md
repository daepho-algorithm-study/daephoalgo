# 🧩 문제

## [완전 이진 트리 - 9934](https://www.acmicpc.net/problem/9934)

### 💡 아이디어 / 접근법
완전 이진 트리이므로 노드의 개수는 2^k - 1개이다.
그리고 중위순회로 여행지를 방문한다.
처음과 끝을 정하고 각 깊이에 중간 값을 삽입하고 왼쪽 오른쪽으로 재귀를 호출 하여준다. 
정답 배열을 출력해주면 된다.

### 🤔 후기
깊이 마다 값을 저장해야 풀 수 있는 문제
