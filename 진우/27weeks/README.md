# 🧩 문제

## [국회의원 선거 - 1417](https://www.acmicpc.net/problem/1417)

### 💡 아이디어 / 접근법
다솜이 보다 표가 많은 후보를 우선적으로 걸러내야 하기 때문에 우선순위큐를 사용한다.
다솜이를 제외한 모든 표들을 우선순위 큐에 집어 넣고, 우선순위 큐의 top의 값이 다솜이의 득표 수 보다 크다면
-1을 하고 다시 우선순위 큐에 넣어준다. 그리고 매수를 했으니 다솜이의 득표 수를 1올려준다.
이걸 반복하면서 매수한 사람들 수를 따로 저장하여 올려주고 출력 해준다.

### 🤔 후기
처음에는 다솜이의 득표 수와 다른 후보의 득표 수를 빼고 더하면서 풀어나가려 했는데 그렇게 하니까 다음 후보에게 표를 뺐으면 이기는 경우와 득표 동점 등의 변수가 생겨서 우선순위큐를 활용하여 후보 득표 1개씩 매수하는 방법으로 풀었다,

# 🧩 문제

## [트럭 주차 - 2979](https://www.acmicpc.net/problem/2979)

### 💡 아이디어 / 접근법
트럭이 들어오는 시간과 나가는 시간을 1과 -1을 pair로 저장한다. 그 후 시간을 기준으로 정렬을 하고,
이전 시간과 현재 시간의 차이 만큼 시간을 계산하고 cnt에 pair의 second를 계속 더해준다. 그렇게 되면 해당 시간에 존재하는 트럭의 갯수를 다시 곱하여 요금을 계산할 수 있다. 

### 🤔 후기
시뮬레이션 상황에 맞춰 직관적으로 코드를 작성하려다 보니 생각보다 어려웠다.
