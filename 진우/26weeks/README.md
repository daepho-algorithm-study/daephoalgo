# 🧩 문제

## [스위치 켜고 끄기 - 1244](https://www.acmicpc.net/problem/1244)

### 💡 아이디어 / 접근법
스위치는 1번부터 시작하기 때문에 0번 인덱스에 0을 넣어주고 1번 인덱스부터 시작
성별이 남자일 때는 반복문을 활용하여 각 인덱스의 배수의 값을 바꿔준다. 절대값을 활용하여 -1을 해주면 항상 0과 1로만 가능
성별이 여자일 때는 왼쪽과 오른쪽 변수를 나눠서 감소 증가 시켜 양쪽의 값이 같은지 확인하고 같은 부분을 범위로 반복문을 활용하여 값을 바꿔준다.
그리고 20줄 마다 줄 바꿈을 해줘야 하므로 20으로 나누어 떨어지면 줄 바꿈을 해준다.

### 🤔 후기
각 성별에 따라 행동이 다르므로 잘 구분해서 구현해주면 되었다.

# 🧩 문제

## [프린터 큐 - 1966](https://www.acmicpc.net/problem/1966)

### 💡 아이디어 / 접근법
중요도가 있기 때문에 우선순위 큐를 활용한다. 그리고 확인을 위해 큐를 하나 더 사용한다.
현재 큐의 값이 우선 순위 큐의 top과 같다면 순서를 증가시키고, 우선순위 큐의 top 값을 빼준다.
아니라면 큐의 front의 값을 뒤로 보낸다.
마지막으로 현재 문서가 가지고 있는 인덱스 값과 내가 찾고자 한 M의 값이 같다면 순서를 출력해준다. 

### 🤔 후기
우선순위 큐를 활용해야겠다고 생각은 했지만, 큐 하나를 더 사용해서 비교하는 것을 셍각하지 못 했다.

# 🧩 문제

## [돌 게임 - 9655 ](https://www.acmicpc.net/problem/9655) 

### 💡 아이디어 / 접근법
1개와 3개만을 가져갈 수 있고 항상 최적의 게임을 하기 때문에  dp를 활용하여 풀 수 있다.
1개만 가져가는 경우와 3개만 가져가는 경우는 dp[i - 1]과 dp[i - 3]이므로 
이 두 값에서 1턴을 더 소모했을 때 더 작은 값을 선택해주면 된다.
그리고 항상 상근이가 먼저 시작하게 되므로 그 선택된 값이 짝수면 창영이의 승리, 홀수이면 상근이의 승리가 된다.
 
### 🤔 후기
이번 문제는 항상 최적의 게임을 진행하기 때문에 돌의 개수가 홀수와 짝수일 때로 나누어서 출력만 해줘도 정답이지만, dp로도 풀이 할 수 있다고 하여 dp로 풀어보았다.
